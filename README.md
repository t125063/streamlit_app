# 日本の人口推移可視化アプリ

## 1. アプリの概要・目的
本アプリは、日本の深刻な社会問題である「少子高齢化」と「人口減少」の現状を直感的に理解することを目的としたWebアプリです。
政府統計（e-Stat）のオープンデータを活用し、以下の2つの視点から可視化を行っています。
* **人口ピラミッド**: 特定の年における男女別・年齢別の人口構成を確認
* **長期推移**: 過去から現在にかけての総人口・男女別人口のトレンドを確認

## 2. データについて
本アプリでは、e-Stat（政府統計ポータルサイト）にて公開されている以下のデータを使用しています。

* **政府統計名**: 人口推計
* **統計表名**: 年齢（各歳）、男女別人口（各年10月1日現在）－総人口、日本人人口（2000年～2020年）
* **使用データ範囲**: 2005年 ～ 2015年
* **データ取得日**: 2026年1月19日
* **各列の意味**:
    * `age`: 年齢（0歳〜100歳以上）
    * `[年]_Male`: 該当年の男性人口（単位：千人）
    * `[年]_Female`: 該当年の女性人口（単位：千人）

## 3. 授業で扱っていないUI部品
講義資料（第12回）のサンプルコードに含まれていない、以下の3つのUI部品を使用しました。

| 使用したUI部品名 | 選んだ目的（なぜ選んだか） | アプリ内での使用箇所 |
| :--- | :--- | :--- |
| **`st.tabs`** | 「ピラミッド」と「推移」という異なる分析視点を、画面を遷移させずにタブで切り替えて比較できるようにするため。 | メインエリアのグラフ表示部分 |
| **`st.container(border=True)`** | グラフ表示エリアを枠線で囲むことで、アプリのメインコンテンツを視覚的に強調し、デザインを整理するため。 | グラフ全体を囲むコンテナ |
| **`st.divider`** | サイドバー内の「ピラミッド設定」と「長期推移設定」の間に境界線を引き、操作項目を分かりやすく区切るため。 | サイドバーの入力項目の間 |

## 4. ギャラリーの参照
* **参考にしたアプリのタイトル**: Streamlit Gallery「Streamlit ECharts Demo」
* **参考にした点**: グラフと設定項目が整理されたモダンなレイアウトや、ユーザーの操作（スライダー等）に対してグラフが即座に反応するようなモノを参考にしました。
* **自分のアプリへの反映**: Plotlyの `Bar` グラフを用いて、サイドバーのスライダーと連動して動的に形状が変化する対話的なピラミッドを実装しました。
